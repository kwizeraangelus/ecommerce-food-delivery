## Project Overview

This project is a Food Delivery Mobile Application UI/UX design created using Figma. The system allows users to browse restaurants, view menus, place food orders, and track deliveries in a simple and user-friendly interface.

The goal is to apply UI/UX design principles, wireframing, prototyping, and accessibility concepts to build a modern and responsive mobile application.
